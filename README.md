# Heart Disease Data Analysis
Exploratory data analysis off the UCI Heart Disease dataset to identify key risk factors associated with heart disease.

This project analyzes the Cleveland Heart Disease dataset from the UCI Machine Learning Repository. The dataset contains 303 patiients and 14 key attributes. The goal is to explore which patient characteristics are most associated with the presence of heart disease.
